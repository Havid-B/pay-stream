## Description

Backend Service for TipStream to enable upload of videos and metadata to decentralized storage using @spheron/storage sdk

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ ysrn start

# watch mode
$ ysrn dev

# production mode
$ ysrn start:prod
```
